# TimedQuiz
A timed quiz on JS basics

## Description
This site runs a quiz on JS basics. The quiz is timed and incorrect answers are penalized with subtraction of time. At the end of the quiz, the user's score and initials are recorded.

## Installation
n/a

## Usage
This site can be used to take a quiz on JS basics.
